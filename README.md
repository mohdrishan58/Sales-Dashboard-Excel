# Sales Dashboard - Excel

An interactive sales dashboard built in Microsoft Excel

## Dashboard Preview
![Dashboard](Dashboard.png)

## What's Inside
- Sales by City
- Sales by Sales Rep
- Product Wise Sale
- Month by Sales
- Sales by Category

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers (Month filter)

## How to Use
1. Download Sales_data1.xlsx
2. Open in Microsoft Excel
3. Click any month in the slicer to filter all charts
